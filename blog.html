What is the difference between localStorage and sessionStorage?

SessionStorage is similar to localStorage. The difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends.

What is the difference between global scope and block scope?

A global variable has global scope. A global variable is accessible from anywhere in the code, whereas a block scoped variable means that the variable defined within a block will not be accessible from outside the block.

How does event loop work in JavaScript?

The event loop simply checks the call stack, and if it is empty (which means there are no functions in the stack) it takes the oldest callback from the callback queue and pushes it into the call stack which eventually executes the callback.

How many ways we can get undefined in JavaScript?

There are eight ways we can get undefined. Variable that is not initialized, function with no return, parameter that is not passed, if return has nothing on the right side, property that doesn't exist on an object, accessing array elements outside of the index range, deleting an element inside an array, and set a value directly to undefined will give undefined.
